Assignment one review





I submitted my first assignment. I was really proud of my wireframes because I spent a long time exploring both options for Calico Air. It has made me realise that it is really hard to achieve a good and intuitive interface, especially when there is a lot of information that requires the user to fill out. I explored the relationship between the intrinsic and extrinsic motivations, and how that affected user flow.

It does not matter what task the user is doing, even if it is travelling overseas for a holiday, the user cannot stay intrinsically motivated unless key touch points such as ‘progress bars’ in a form filling setting.

I was really happy with my grade and will try to keep backing my assignments with correct UX practices such as researching, and block framing.
